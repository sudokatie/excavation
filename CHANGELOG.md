# Changelog

## 0.1.0 - 2026-02-01

Initial release.

### Features
- Grid-based puzzle gameplay
- Physics system with gravity and rolling
- 5 playable levels
- Player movement, digging, and boulder pushing
- Gem collection and exit mechanics
- Time-based scoring with bonuses
- Pause, restart, and level select

### Technical
- 95 tests passing
- TypeScript with strict mode
- Canvas-based rendering
